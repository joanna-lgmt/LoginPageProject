# Swasthyabot Login Page

This project contains a simple and responsive login page built using HTML, CSS, and JavaScript for the Swasthyabot medical help AI agent. It includes username and password fields with basic client-side validation. The design is clean, user-friendly, and suitable as a starter template for web applications or learning front-end development.

## How to Use
1. Clone or download this repository to your local machine.
2. Open `index.html` in any web browser to view and test the Swasthyabot login page.
